# Project Dashboard

## Overview
This repository hosts a Next.js dashboard that surfaces development system status by digesting GitHub data and presenting it through a structured UI. It is designed for maintainability, extensibility, and clear separation between the data ingestion layer and the presentation layer.

## Prerequisites
Before you begin, ensure the following are installed on your machine:

- [Node.js 18+](https://nodejs.org/) (includes npm)
- [Git](https://git-scm.com/)
- A GitHub account with access to the repositories you want to monitor
- A GitHub personal access token with appropriate scopes (`repo`, `read:org`, etc.)
