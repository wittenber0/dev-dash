# Project Dashboard

## Overview
This repository hosts a Next.js dashboard that surfaces development system status by digesting GitHub data and presenting it through a structured UI. The goal is to provide visibility into critical workflows while remaining easy to extend through reusable layouts, components, and navigation structures.

## Prerequisites
Before you begin, ensure the following are installed:

- [Node.js 18+](https://nodejs.org/) (includes npm)
- [Git](https://git-scm.com/)
- A GitHub account with access to relevant repositories
- A GitHub personal access token with scopes such as `repo`, `read:org`, and any other scopes needed to read organization or team data
